## Console
```
Console - Shipped console (interactive command and response)
------------------------------------------------------------
Run Shipped in console mode to interact directly with Shipped. The Shipped console supports command completion (tab to
finish a command or argument) and history (up arrow to display, edit, and rerun previous commands).

Type quit or exit to exit the Shipped console and return to your OS.
```
